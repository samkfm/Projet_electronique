Alors ici on mettera toutes les explications du programme s'il y a besoin voilà voilà !

Pour l'écran : 
GND -> GND 
VCC -> 5V 
SCL -> A5 
SDA -> A4

Pour les capteurs (c'est la même chose pour tous) : 
GND -> GND 
VDD -> 5V 
VIN -> 3.3V 
SCL -> A5 
SDA -> A4
GPIO0/CE -> 3.3V 

Pour le buzzer : 
GND -> -
D8 -> + 

Pour le bargraphe :
côté blanc = - et coté écriture = +
Du coté + on met des résistances 220 kohm
chaque LED a sa patte arduino (exemple D2 = LED 1)
